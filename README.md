# Lucky Spin

A simple web app that generates random daily suggestions to help users decide what to do.

Click "SPIN!" to randomly select the tasks for today, and display the corresponding pictures and descriptions.

## File Structure

- `index.html`: Page structure
- `style.css`: Page style
- `script.js`: Logic for random tasks, page switching and image requests

## Usage Method

Just open in your browser and it will run.

## Explanation

- The home page will request the Meme API to load images.
- The result page will request Unsplash images based on the task keywords.
- You can re-spin anytime if you don't like the result.
